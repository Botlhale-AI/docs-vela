Agent Details Management

The Agent Details section within the Vela platform provides a comprehensive view and management capabilities for all agents within your organization. Here you can view agent information, add new agents, edit existing ones, and perform bulk actions.


Fig 1: The main Agent Details screen.

Accessing Agent Details​

To access the Agent Details page:

  1. Navigate to the main left-hand sidebar menu.
  2. Click on Agents.
  3. Select Agent Details from the sub-menu that appears.

Interface Overview​

The Agent Details page consists of several key components:

1. Agent Table​

This is the central part of the screen, displaying a list of all agents with relevant information organized into columns:

  • Checkbox: Used for selecting one or multiple agents for bulk actions. There's also a master checkbox in the header row to select/deselect all visible agents.
    Fig 2: Header checkbox for selecting all agents.
  • Name: The agent's name.
  • Email: The agent's email address.
  • Department: The department the agent belongs to.
  • Team: The specific team the agent is assigned to.
  • Voice Profile Status: Indicates the status of the agent's voice profile setup (e.g., "Waiting"). This may include an interactive element (like a toggle) for status management or viewing details. Fig 3: Voice Profile Status indication.
  • Actions: Provides icons for performing actions on individual agents:
    • Edit (Pencil Icon): Modify the agent's details.
    • Delete (Trash Can Icon): Remove the agent from the system.
    Fig 4: Edit and Delete actions for an agent.

2. Toolbar Actions​

Located above the agent table, these buttons allow you to manage the view and data:


Fig 5: Toolbar actions - Search, Sort, Filter, Export.

  • Search: Allows you to find specific agents by typing keywords (e.g., name, email).
  • Sort: Options to sort the agent list based on different columns (e.g., alphabetically by Name, by Department).
  • Filter: Apply filters to narrow down the list of agents based on specific criteria (e.g., show only agents from a particular Team or Department).
  • Export: Download the currently displayed agent list, potentially as a CSV or other file format.

3. Bulk Action Buttons​

These buttons appear above the table after you select one or more agents using the checkboxes.

  • Reassign: Allows you to reassign selected agents, likely to a different Team or Department.
    Fig 6: The "Reassign" button appears after selecting agents.

4. Add Agent Button​

Located in the top-right corner, this button initiates the process of adding new agents to the system.


Fig 7: The "+ Add Agent" button.

Managing Agents​

Adding a New Agent​

Clicking the + Add Agent button opens a modal window with two options:

  1. Single Upload: Add agents one by one by manually entering their details.

    • Name: Enter the agent's full name.
    • Email: Enter the agent's email address.
    • Team: Assign the agent to a specific team.
    • Click Save to add the agent.
      Fig 8: Adding a single agent.

  2. Batch Upload: Upload a list of agents using a CSV file.

    • Ensure your CSV file matches the required format (an example or template might be provided).
    • Drag and drop your CSV file onto the designated area or click to browse and select the file.
    • Click Save to process the batch upload.
      Fig 9: Adding multiple agents via batch upload.

    Info: When adding agents, Vela/Bothale AI may prompt them for a voice sample later to create a unique ID for voice-based routing and analysis.

Editing an Agent​

  1. Locate the agent you wish to edit in the table.
  2. Click the Edit (Pencil) icon in the Actions column for that agent's row.
  3. A modal or form will appear allowing you to modify the agent's details (Name, Email, Department, Team, etc.).
  4. Save the changes.

Deleting an Agent​

Caution: Deleting an agent is typically permanent and may affect historical reporting.

  1. Locate the agent you wish to delete in the table.
  2. Click the Delete (Trash Can) icon in the Actions column for that agent's row.
  3. Confirm the deletion when prompted.

Performing Bulk Actions (Example: Reassign)​

  1. Select the agents you want to perform an action on by clicking the checkboxes next to their names. You can select all visible agents by clicking the checkbox in the table header.
  2. Once agents are selected, bulk action buttons like Reassign will appear above the table.
  3. Click the desired bulk action button (e.g., Reassign).
  4. Follow the prompts to complete the action (e.g., select the new Team/Department for reassignment).
